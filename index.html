About Me
Web Dev Is My Passion

A programmer is a person whose job is to develop software or applications. When creating a program, there is a flow chart that must be considered and then translated into computer language using certain code so that it becomes a command for the computer to carry out certain tasks. Programming languages, such as CSS, SQL, HTML, JavaScript, PHP, XML, Python, and so on have become familiar things for a programmer. Now, when the program has been created, the programmer must ensure that the program operates correctly. The programmer will carry out various tests by predicting each case that will occur because the computer will not be able to operate when there is an error. Therefore, programmers must be able to be friendly with errors. Because every time you create a program, there will always be trial and error.
